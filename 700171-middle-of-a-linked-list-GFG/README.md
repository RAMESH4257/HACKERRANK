# [Middle of a Linked List](https://www.geeksforgeeks.org/problems/finding-middle-element-in-a-linked-list/1)
## Easy
You are given the head of a linked list, You have to return the value of the middle node of the linked list. 

If the number of nodes is odd, return the middle node value.
If the number of nodes is even, there are two middle nodes, so return the second middle node value.

Examples:
Input:    
Output: 3Explanation: The given linked list is 1-&gt;2-&gt;3-&gt;4-&gt;5 and its middle is 3.   
Input:   Output: 7 Explanation: The given linked list is 2-&gt;4-&gt;6-&gt;7-&gt;5-&gt;1 so, there are two middle node 6 and 7, return the second middle node as 7.   
Constraints:1 ≤ no. of nodes ≤ 1051 ≤ node-&gt;data ≤ 105

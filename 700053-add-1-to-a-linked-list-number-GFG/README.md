# [Add 1 to a Linked List Number](https://www.geeksforgeeks.org/problems/add-1-to-a-number-represented-as-linked-list/1)
## Medium
You are given head of a linked list where each node contains a single digit. The digits together represent a number formed by concatenating the node values in order. Add 1 to this number and return the head of the modified linked list.
Examples :
Input: Head: 4-&gt;5-&gt;6
Output: 457Explanation: 4-&gt;5-&gt;6 represents 456 and when 1 is added it becomes 457. 

Input: Head: 1-&gt;2-&gt;3
Output: 124 Explanation:  1-&gt;2-&gt;3 represents 123 and when 1 is added it becomes 124. 
Input: Head: 0-&gt;0-&gt;1
Output: 002
Constraints:1&nbsp;≤ len(list)&nbsp;≤&nbsp;100&nbsp;≤ list[i]&nbsp;≤&nbsp;9

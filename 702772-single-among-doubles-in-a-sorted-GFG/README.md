# [Single Among Doubles in a Sorted](https://www.geeksforgeeks.org/problems/find-the-element-that-appears-once-in-sorted-array0624/1)
## Medium
Given a sorted array arr[]. Find the element that appears only once in the array. All other elements appear exactly twice.&nbsp;
Examples:
Input: arr[] = [1, 1, 2, 2, 3, 3, 4, 50, 50, 65, 65]
Output: 4
Explanation: 4 is the only element that appears exactly once.
Input: arr[] = [5]
Output: 5
Input: arr[] = [1, 2, 2, 3, 3]
Output: 1
Constraints:1 ≤ arr.size() ≤ 2 * 1061 ≤ arr[i] ≤ 106

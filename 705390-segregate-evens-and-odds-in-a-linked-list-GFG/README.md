# [Segregate Evens and Odds in a Linked List](https://www.geeksforgeeks.org/problems/segregate-even-and-odd-nodes-in-a-linked-list5035/1)
## Medium
Given a link list, modify the list such that all the even numbers appear before all the odd numbers in the modified list. The order of appearance of numbers within each segregation should be the same as that in the original list.
NOTE:&nbsp;Don't create a new linked list, instead rearrange the provided one.
Examples:
Input: Linked list: 17-&gt;15-&gt;8-&gt;9-&gt;2-&gt;4-&gt;6
Output: 8-&gt;2-&gt;4-&gt;6-&gt;17-&gt;15-&gt;9Explaination: 8,2,4,6 are the even numbers so they appear first and 17,15,9 are odd numbers that appear later.
Input: Linked List: 1 -&gt; 3 -&gt; 5 -&gt; 7
Output: 1-&gt;3-&gt;5-&gt;7Explaination: There is no even number. So no need for modification.
Constraints:1 ≤ size of linked list ≤ 1051 ≤ Each element of the list ≤ 105
